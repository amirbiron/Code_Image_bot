1. **חלון macOS מלא** עם:

   - שורת כותרת אפורה
   - רקע כהה לקוד
   
2. **רקע צבעוני מדהים**:
   - גרדיאנט אלכסוני
   - צבעים שונים לכל נושא
   
3. **צללים ואפקטים**:
   - צל רך מתחת לחלון
   - פינות מעוגלות
   - נראה מקצועי ביותר!

## 🆚 ההבדלים בין הגרסאות

### `code_image_bot_macos.py` ⭐ (זו שאתה רוצה!)
- חלון macOS מלא עם כפתורים
- רקעים צבעוניים עם גרדיאנט
- בדיוק כמו בתמונה שהראית

### `code_image_bot.py` (בסיסי)
- רק קוד פשוט עם syntax highlighting
- ללא חלון או רקע מיוחד

### `code_image_bot_enhanced.py` (מתקדם כללי)
- כמה אפקטים נוספים
- אבל לא דומה לתמונה שהראית

## ⚙️ הגדרות ב-Render

**Start Command:**
```bash

```

**Environment Variables:**
- `TELEGRAM_BOT_TOKEN` = הטוקן שלך

זהו! פשוט ונקי.
